### 1. Always Preview First
```
GOOD: "Preview 20 examples before applying to all products"
BAD:  "Apply to all 4,000 products immediately"
```

### 2. Start with Small Test Batches
```

```

- Monitor conversion rate improvements
- Review customer feedback
- Iterate based on data

## Troubleshooting

### "Connection failed" Error
**Fix**:
1. Check WooCommerce is installed and activated
2. Verify REST API is enabled
3. Ensure your consumer key has Read/Write permissions
4. Check for WordPress security plugins blocking API access

### "Too many requests" Error
**Fix**:
1. WooCommerce has rate limits (usually 100 requests/min)
2. Tandril automatically handles this with batching
3. For very large operations, commands run in background

### Products not syncing
**Fix**:
1. Check "Last Synced" timestamp in Platforms page
2. Click "Sync Now" to force manual sync
3. Check WooCommerce error logs for API issues
